# ROTEIRO_-DE_-AULA_FRA-ES
Este repositório reúne planos de aula e materiais pedagógicos para o 6º ano, integrando o uso ético e criativo da Inteligência Artificial como apoio ao ensino de frações. O objetivo é oferecer recursos visuais, atividades interativas e estratégias diferenciadas que promovem a compreensão matemática e o desenvolvimento digital dos alunos.
